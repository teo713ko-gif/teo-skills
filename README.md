# 태오의 실행 비즈니스 — 무료 스킬 모음

유튜브 [태오의 실행 비즈니스](https://www.youtube.com/@teoactbusiness) 영상에서 소개한 클로드(Claude) 스킬을 무료로 나눕니다.
1인 기업을 시작하고 굴리는 데 필요한 것들을 하나씩 추가합니다.

## 스킬 목록

| 스킬 | 하는 일 | 소개 영상 |
|---|---|---|
| [1인사업-설계](./1인사업-설계/) | 사명 찾기 4질문 인터뷰 → 시장 검증 → 팔 수 있는 오퍼 설계 → 첫 7일 실행 체크리스트까지 한 번에 | 준비 중 |

## 설치 방법

### 방법 1 — 클로드 코드에 붙여넣기 (영상에서 보여드린 방식 · 제일 쉬움)

1. 이 페이지 주소를 복사합니다
   ```
   https://github.com/teo713ko-gif/teo-skills
   ```
2. 클로드 코드 채팅창에 붙여넣고 이렇게 말합니다
   ```
   이 저장소의 1인사업-설계 스킬을 복제해서 적용해줘
   ```
3. 끝입니다. 클로드가 알아서 내려받아 설치해요

**설치 위치 참고**
- 그냥 설치하면 **홈 폴더**(유저 스코프, `~/.claude/skills/`)에 깔려서 모든 프로젝트에서 쓸 수 있어요
- 지금 열려 있는 **프로젝트에만** 쓰고 싶으면 이렇게 말하면 됩니다
  ```
  이 프로젝트의 .claude/skills/ 에 설치해줘
  ```

### 방법 2 — 클로드 앱 (데스크톱)

1. [1인사업-설계-스킬.zip](./1인사업-설계/1인사업-설계-스킬.zip) 을 내려받습니다
2. 클로드 앱 → 설정(Customize) → 스킬(Skills) → 스킬 업로드에서 zip을 올립니다

## 사용 방법

설치했으면 새 창을 열고 아래 중 아무거나:
- `/1인사업-설계` 라고 치기
- 또는 그냥 자연어로 — **"내 사업 설계 좀 해줘"**, "내 경험으로 뭘 팔 수 있을까?"

질문 4개에 답하면 → 시장 검증 → 사명 한 문장 → 오퍼 설계 → 첫 7일 실행 체크리스트가 나옵니다.
AI가 뽑아주는 건 간단해요. 실행은 여러분 몫입니다 — 실행하고 나서 영상에 댓글 남겨주세요, 제가 직접 봅니다.
