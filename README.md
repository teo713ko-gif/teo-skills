# 태오의 실행 비즈니스 — 무료 스킬 모음

유튜브 [태오의 실행 비즈니스](https://www.youtube.com/@teoactbusiness) 영상에서 소개한 클로드(Claude) 스킬을 무료로 나눕니다.
1인 기업을 시작하고 굴리는 데 필요한 것들을 하나씩 추가합니다.

## 스킬 목록

| 스킬 | 하는 일 | 소개 영상 |
|---|---|---|
| [1인사업-설계](./1인사업-설계/) | 사명 찾기 4질문 인터뷰 → 시장 검증 → 팔 수 있는 오퍼 설계까지 한 번에 | 준비 중 |

## 설치 방법

### 클로드 앱 (데스크톱)
1. 이 저장소에서 **Code → Download ZIP** 을 눌러 내려받고 압축을 풉니다
2. 클로드 앱 → 설정(Customize) → 스킬(Skills) → 스킬 업로드
3. 원하는 스킬 폴더(예: `1인사업-설계`)를 zip으로 압축해서 업로드합니다
4. 채팅에서 `/1인사업-설계` 라고 치거나, "내 경험으로 뭘 팔 수 있을까?" 라고 물어보면 시작됩니다

### 클로드 코드 (Claude Code)
1. 스킬 폴더를 내 프로젝트의 `.claude/skills/` 안에 복사합니다
   (예: `.claude/skills/1인사업-설계/SKILL.md`)
2. 새 세션에서 `/1인사업-설계` 로 실행합니다

## 이런 분께

- 클로드는 쓰고 있는데 내 사업에 어떻게 붙일지 막막한 분
- 뭘 팔아야 할지, 누구한테 팔아야 할지부터 막힌 분
- 혼자 12시간 일하다 반복 업무를 넘기고 싶은 분
